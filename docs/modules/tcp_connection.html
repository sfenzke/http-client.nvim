send (host, port, data, callback)
Sends data to a specified host and port using TCP. This function resolves the hostname, attempts connections sequentially, and sends data upon successful connection.

Parameters:

  • host string The target hostname.
  • port number The target port number.
  • data string The data to be sent.
  • callback function A function to handle the response or error. It receives two parameters: (error: string|nil, response: string|nil).
