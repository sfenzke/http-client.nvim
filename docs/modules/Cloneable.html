Module Cloneable

A module that provides clonable functionality for objects.

This module allows objects with a clone method to be cloned, including objects with metatables that can also be cloned recursively.

Functions

clone () Clones the current object, including its metatable and all clonable fields.


Functions

clone ()
Clones the current object, including its metatable and all clonable fields. The cloned object will be a shallow copy, but fields that are clonable will be cloned recursively.

Returns:

    Cloneable A new object that is a clone of the current object.
